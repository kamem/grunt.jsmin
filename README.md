grunt.jsmin
===========

jsフォルダに入っているjsに同ファイル名.txtをコピーライトとして追加した.minファイルを作る

仕様
------
1. pluginNameに圧縮したいjsの名前を指定するとそのファイルがjsフォルダの同ファイル名.txtをコピーライトとしてminフォルダに同ファイル名.min.jsファイルが制作される

使い方
------
grunt --pluginName=jquery.sample
